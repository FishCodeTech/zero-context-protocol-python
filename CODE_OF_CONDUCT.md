# Code Of Conduct

This project adopts the Contributor Covenant Code of Conduct.

## Our Standards

Examples of behavior that contributes to a positive environment:

- being respectful and professional
- accepting constructive feedback
- focusing on technical issues rather than people
- documenting claims, benchmarks, and tradeoffs clearly

Examples of unacceptable behavior:

- harassment or personal attacks
- publishing private information without permission
- bad-faith disruption or abusive conduct

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and
other contributions that violate this Code of Conduct.

## Reporting

Report conduct issues through the security/contact route documented in
[`SECURITY.md`](SECURITY.md).
